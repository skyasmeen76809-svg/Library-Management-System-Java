# Library Management System (Java)

A console-based Library Management System developed using Core Java. 
This project demonstrates object-oriented programming (OOP) concepts 
and basic library operations through a simple, user-friendly console interface.

## Features
- Add new books to the library
- View available books
- Issue books to users
- Return issued books
- Simple and menu-driven console interface

## Technologies Used
- Java (Core Java)
- Object-Oriented Programming (OOP)

## Project Structure
- Mainclass.java – Entry point of the application
- Book.java – Represents book details and properties
- BookService.java – Handles library operations and logic

## How to Run
1. Clone or download the repository
2. Compile all `.java` files
3. Run `Mainclass.java`
4. Follow the console menu instructions

## Purpose
This project was created for learning and practice purposes to 
strengthen Java fundamentals and OOP concepts.
